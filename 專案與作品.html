專案與作品

⼈．天堂－Playyounger

專案目標：我們意識到許多長者面臨著健康問題和社交孤立的困境，本平台是網頁結合傳感器遊戲與會員、排行榜、數據分析等功能，期待改善長者社交與運動習慣。

專案技術：

1.掌握 FLASK 框架的使用，包括前後端的完整開發流程。

2.深入了解多軸感應器 SDK 的技術細節和應用場景。

3.學習使用 Web Socket 和 HTTP 技術進行資料的即時傳輸和儲存。

4.掌握 echarts 圖表庫的使用，以便更好地呈現數據分析結果。

5.加強跨領域（如遊戲、硬體技術、數據分析等）的協調與整合能力。

6.透過各式測試的回饋和行為數據，持續優化網頁介面、功能和互動設計。

7.設計並實施有效的使用者測試，收集並分析數據以指導設計決策。

8.提高對使用者需求的敏感性，設計更具吸引力和易用性的遊戲特性。

負責工作：負責處理echarts連接資料庫呈現數據分析、撰寫感測器轉換遊戲操作的程式、架設GCP伺服器測試網頁架站。

專案PPT

社群平台爬蟲與數據分析

專案目標：社群平台是近年廣告投放的重要渠道，我們以「籃球」為主題，藉由爬蟲去獲取twitter、IG、FB的貼文流量與用戶情緒，分析出最適合投放相關題材的社群帳號與主題建議。

專案技術：

1.Python 程式設計：掌握 Python 語言的基礎語法和相關函式庫，用於開發爬蟲程式和數據處理。

2.爬蟲技術：了解爬蟲的基本原理和常見的爬蟲工具，並能夠運用相關技術爬取 PTT 和 Dcard 上的相關資料。

3.數據分析與視覺化：學習如何處理和分析爬取的資料，並運用視覺化工具（如 echarts）將分析結果呈現出來，以便進行資料可視化和分析報告的製作。

4.團隊合作與專案管理：通過與團隊成員合作，共同設計爬蟲程式、進行數據分析和視覺化呈現，並學習如何有效地分工合作、專案管理和溝通協作。
